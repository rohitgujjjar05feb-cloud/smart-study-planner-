# 📘 Smart Study Planner

A modern web-based study planner to manage daily tasks efficiently.

##  Features
- Add, Edit, Delete Tasks
- Mark tasks as completed
- Modal-based UI for task input
- Filter tasks (All / Completed / Pending)
- Circular progress tracking
- Dashboard with task statistics
- LocalStorage support (data persists after refresh)

##  Tech Stack
- HTML
- CSS
- JavaScript
